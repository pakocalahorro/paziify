# 🛑 PROTOCOLO DE TRABAJO PAZIIFY

¡Bienvenido! Para mantener la App con **Zero Defectos**, seguimos este protocolo obligatorio:

1. 🌿 **Rama**: Nunca trabajes en `main`. Crea una Rama Borrador (`git checkout -b feature/nombre`).
2. 📸 **Fotos**: No rompas la estética. El robot sacará fotos (Snapshots).
3. ✅ **Robot**: No se publica nada sin el Check Verde de GitHub Actions.
4. 👑 **CEO**: El OK final de la versión es de **Paco**.

---
### 🛠️ Comandos de Emergencia
* `npm run reset:paziify`: Borra tus cambios actuales y vuelve al punto de salud total.
* `npm test`: Ejecuta los robots de lógica y fotos manualmente.
